# NSK_Pepper_RadioExecise

■笑いヨガのジェスチャー
* 笑いヨガのジェスチャーを配布致します。
* 配布するジェスチャーは全6個
  
　　1. アロハ笑い  
　　2. １メートル笑い  
　　3. ミルクセーキ笑い  
　　4. 携帯電話笑い  
　　5. 請求書笑い  
　　6. ホ・ホ、ハ・ハ・ハ  

======================

■ ボックスライブラリの追加手順  
　1. 本ページより、「***.cbl」をダウンロードする  
　2. コレグラフ内から、「ボックスライブラリを開く」を選択  
　3. ダウンロードした「***.cbl」を選択し、追加する
